## Namespace

Robotics namespace URI:

## Demo Robot

The demo robot is a simulated 6-axis industrial robot.

## Axis Mapping

| Axis | Meaning |
| --- | --- |
| S-axis | Base rotation |
| L-axis | Lower arm forward/backward |
| U-axis | Upper arm up/down |
| R-axis | Arm rotation |
| B-axis | Wrist bend |
| T-axis | Tool flange rotation |
