# Robotics OPC UA Model Generated

This project contains generated code from the official OPC UA DI and OPC UA Robotics NodeSets.

The files are generated by:

```powershell
.\tools\modelcompiler\generate-robotics-model.ps1
```

Generated files should not be manually edited. Update the official NodeSet inputs or generation process, then regenerate the output.

Robotics generated code depends on DI generated code. For example, Robotics classes can reference DI state classes such as `ComponentTypeState`. The project therefore intentionally includes generated C# files from both `generated/opcua-di/` and `generated/opcua-robotics/` for now.

IA and Machinery are still future optional integrations and are not included in this generated project yet.

The project exists so the reference server can later instantiate official OPC UA Robotics model types without mixing generated code into hand-written server code.
