# Robotics OPC UA Robotics Model Generated

This project contains generated code from the official OPC UA Robotics NodeSets.

The files are generated by:

```powershell
.\tools\modelcompiler\generate-robotics-model.ps1
```

Generated files should not be manually edited. Update the official NodeSet inputs or generation process, then regenerate the output.

The project exists so the reference server can later instantiate official OPC UA Robotics model types without mixing generated code into hand-written server code.
